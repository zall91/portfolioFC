Connecte-toi avec ton compte GitHub pour modifier ton portfolio.

1. Va sur github.com/settings/tokens
2. Generate new token → Fine-grained
3. Nom : admin-portfolio — Expiration : 90 jours
4. Repository access → Only select repositories → ton dépôt
5. Permissions → Contents → Read and write
6. Copie le token ici. Il reste dans ton navigateur uniquement (sessionStorage).
