⚡ Admin Portfolio

Connecte-toi avec ton compte GitHub pour modifier ton portfolio.

Étapes (2 minutes) :
1. Va sur github.com/settings/tokens
2. Clique sur Generate new token → Fine-grained token
3. Nom : admin-portfolio
4. Expiration : 90 jours (ou plus)
5. Repository access → Only select repositories → choisis ton dépôt portfolio
6. Permissions → Repository permissions → Contents → Read and write
7. Clique Generate token et copie-le ici

Sécurité : Ce token reste uniquement dans ton navigateur (sessionStorage). Il disparaît quand tu fermes l'onglet.
