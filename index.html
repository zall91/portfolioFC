Infrastructure

Cluster Proxmox VE

Contexte : Projet réalisé en TP dans le cadre du BTS SIO SISR.

Objectif : Mettre en place un environnement de virtualisation haute disponibilité pour héberger plusieurs machines virtuelles de production.

Technologies utilisées :

  • Proxmox VE 8.x
  • Stockage partagé NFS (Debian)
  • Cluster à 2 nœuds avec migration à chaud
  • Sauvegarde automatisée (PBS)

Résultat : Infrastructure fonctionnelle permettant la migration de VM sans interruption de service.

Sécurité

Firewall pfSense

Contexte : Sécurisation du réseau d'un labo technique.

Objectif : Déployer un pare-feu open source pour filtrer le trafic, créer des tunnels VPN et mettre en place un portail captif.

Technologies utilisées :

  • pfSense CE
  • VPN IPSec site à site
  • Squid + SquidGuard (filtrage URL)
  • Portail captif avec authentification RADIUS

Résultat : Réseau sécurisé avec séparation des flux et accès contrôlé.

Réseau

Segmentation VLAN

Contexte : Projet de TP réseau dans le cadre du module SI2.

Objectif : Segmenter un réseau d'entreprise en plusieurs VLANs pour isoler les départements et sécuriser les communications.

Technologies utilisées :

  • Switches Cisco Catalyst (Packet Tracer / matériel réel)
  • VLANs, trunking 802.1Q
  • Routage inter-VLAN (Router on a stick)
  • ACLs pour restriction de flux

Résultat : Réseau segmenté avec politique d'accès entre les départements.

Système

Active Directory

Contexte : Projet de TP système dans le cadre du module SI1.

Objectif : Installer et configurer un domaine Active Directory complet pour gérer les utilisateurs, postes et stratégies de sécurité.

Technologies utilisées :

  • Windows Server 2022
  • AD DS, DNS, DHCP
  • GPO (stratégies de groupe)
  • Gestion des OU et profils itinérants

Résultat : Domaine fonctionnel avec gestion centralisée de 30+ comptes utilisateurs.
